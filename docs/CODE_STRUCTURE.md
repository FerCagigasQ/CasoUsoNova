## 2. Entity: Guarantee (JPA + H2)

### Domain Model

```java
// guarantees-service/src/main/java/com/example/guarantees/domain/Guarantee.java
@Entity
@Table(name = "guarantee")
@Data
@NoArgsConstructor
@AllArgsConstructor
public class Guarantee {
    
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;
    
    @Column(nullable = false, unique = true, length = 50)
    private String guaranteeNumber;
    
    @Column(nullable = false)
    @Enumerated(EnumType.STRING)
    private GuaranteeStatus status;  // ACTIVE, INACTIVE, EXPIRED
    
    @Column(nullable = false, precision = 19, scale = 2)
    private BigDecimal amount;
    
    @Column(length = 3)
    private String currency;  // EUR, USD, etc
    
    @Temporal(TemporalType.TIMESTAMP)
    private LocalDateTime startDate;
    
    @Temporal(TemporalType.TIMESTAMP)
    private LocalDateTime endDate;
    
    @Temporal(TemporalType.TIMESTAMP)
    @CreationTimestamp
    private LocalDateTime createdAt;
    
    @Temporal(TemporalType.TIMESTAMP)
    @UpdateTimestamp
    private LocalDateTime updatedAt;
}

// Enum
public enum GuaranteeStatus {
    ACTIVE,
    INACTIVE,
    EXPIRED
}
```

### Generated SQL (H2)

Cuando Spring Boot inicia con `spring.jpa.hibernate.ddl-auto=update`, Hibernate genera:

```sql
CREATE TABLE GUARANTEE (
    ID BIGINT GENERATED BY DEFAULT AS IDENTITY NOT NULL,
    AMOUNT DECIMAL(19, 2) NOT NULL,
    CREATED_AT TIMESTAMP,
    CURRENCY VARCHAR(3),
    END_DATE TIMESTAMP,
    GUARANTEE_NUMBER VARCHAR(50) UNIQUE NOT NULL,
    START_DATE TIMESTAMP,
    STATUS VARCHAR(20) NOT NULL,
    UPDATED_AT TIMESTAMP,
    PRIMARY KEY (ID)
);

CREATE UNIQUE INDEX UK_GUARANTEE_NUMBER ON GUARANTEE(GUARANTEE_NUMBER);
```

### Mapeo Campos

| Entity Attribute | Database Column | Type | Notes |
|------------------|-----------------|------|-------|
| id | ID | BIGINT | Primary key, auto-generated |
| guaranteeNumber | GUARANTEE_NUMBER | VARCHAR(50) | Unique, business identifier |
| status | STATUS | VARCHAR(20) | Enum: ACTIVE/INACTIVE/EXPIRED |
| amount | AMOUNT | DECIMAL(19,2) | Money: euros, precisión 2 decimales |
| currency | CURRENCY | VARCHAR(3) | ISO 4217: EUR, USD, etc |
| startDate | START_DATE | TIMESTAMP | Fecha inicio de la garantía |
| endDate | END_DATE | TIMESTAMP | Fecha vencimiento |
| createdAt | CREATED_AT | TIMESTAMP | Auditoría: fecha creación |
| updatedAt | UPDATED_AT | TIMESTAMP | Auditoría: fecha última modificación |

## 4. Service Layer (Business Logic)

```java
// guarantees-service/src/main/java/com/example/guarantees/service/GuaranteeService.java
@Service
@Transactional
public class GuaranteeService {
    
    @Autowired
    private GuaranteeRepository repository;
    
    // Create
    public GuaranteeDTO createGuarantee(CreateGuaranteeRequest request) {
        Guarantee guarantee = new Guarantee();
        guarantee.setGuaranteeNumber(generateNumber());
        guarantee.setAmount(request.getAmount());
        guarantee.setCurrency(request.getCurrency());
        guarantee.setStatus(GuaranteeStatus.ACTIVE);
        guarantee.setStartDate(LocalDateTime.now());
        
        Guarantee saved = repository.save(guarantee);  // INSERT
        return mapToDTO(saved);
    }
    
    // Read
    public GuaranteeDTO getGuarantee(Long id) {
        Guarantee guarantee = repository.findById(id)  // SELECT by ID
            .orElseThrow(() -> new NotFoundException("Guarantee not found"));
        return mapToDTO(guarantee);
    }
    
    // List
    public List<GuaranteeDTO> listActiveGuarantees() {
        List<Guarantee> guarantees = repository.findByStatus(GuaranteeStatus.ACTIVE);  // SELECT with filter
        return guarantees.stream()
            .map(this::mapToDTO)
            .collect(Collectors.toList());
    }
    
    // Update
    public GuaranteeDTO updateGuarantee(Long id, UpdateGuaranteeRequest request) {
        Guarantee guarantee = repository.findById(id).orElseThrow();
        guarantee.setAmount(request.getAmount());
        guarantee.setStatus(request.getStatus());
        
        Guarantee updated = repository.save(guarantee);  // UPDATE (H2 detecta cambios)
        return mapToDTO(updated);
    }
    
    // Delete
    public void deleteGuarantee(Long id) {
        repository.deleteById(id);  // DELETE
    }
}
```

### Transaction Management

H2 en memoria maneja transacciones automáticamente:
- `@Transactional`: Spring demarca boundaries de transacción
- `save()` ejecuta INSERT/UPDATE en la transacción
- En commit, cambios se escriben a RAM (no hay disco)
- En rollback, cambios se descartan

## 5. REST Controller

```java
// guarantees-service/src/main/java/com/example/guarantees/controller/GuaranteeController.java
@RestController
@RequestMapping("/api/v1/guarantees")
@Validated
public class GuaranteeController {
    
    @Autowired
    private GuaranteeService service;
    
    @PostMapping
    public ResponseEntity<GuaranteeDTO> create(@RequestBody @Valid CreateGuaranteeRequest request) {
        GuaranteeDTO dto = service.createGuarantee(request);
        return ResponseEntity.status(HttpStatus.CREATED).body(dto);
    }
    
    @GetMapping("/{id}")
    public ResponseEntity<GuaranteeDTO> get(@PathVariable Long id) {
        GuaranteeDTO dto = service.getGuarantee(id);
        return ResponseEntity.ok(dto);
    }
    
    @GetMapping
    public ResponseEntity<List<GuaranteeDTO>> list() {
        List<GuaranteeDTO> dtos = service.listActiveGuarantees();
        return ResponseEntity.ok(dtos);
    }
    
    @PutMapping("/{id}")
    public ResponseEntity<GuaranteeDTO> update(@PathVariable Long id, @RequestBody UpdateGuaranteeRequest request) {
        GuaranteeDTO dto = service.updateGuarantee(id, request);
        return ResponseEntity.ok(dto);
    }
    
    @DeleteMapping("/{id}")
    public ResponseEntity<Void> delete(@PathVariable Long id) {
        service.deleteGuarantee(id);
        return ResponseEntity.noContent().build();
    }
}
```

## 8. Lifecycle & Boot Sequence

```
1. Docker start
   └─ Container created
       └─ JVM started
           └─ Classpath scanning

2. Spring Boot initialization
   ├─ Load application.yml (base config)
   ├─ Load application-docker.yml (override: JDBC URL)
   ├─ Create DataSource bean
   │   └─ Open H2 connection: jdbc:h2:mem:testdb
   ├─ Create H2 schema: public
   └─ Initialize Hibernate

3. Entity scanning & DDL
   ├─ Scan @Entity classes (Guarantee.java found)
   ├─ Build metamodel
   ├─ Run ddl-auto: update
   │   ├─ Check if table GUARANTEE exists → NO
   │   ├─ Generate CREATE TABLE statement
   │   └─ Execute: CREATE TABLE GUARANTEE (...)
   └─ Create indexes (PK, unique constraints)

4. Repository initialization
   ├─ Create GuaranteeRepository bean (proxy)
   └─ Ready for queries

5. Controller initialization
   ├─ Register @RestController
   ├─ Register endpoints: /api/v1/guarantees/*
   └─ Health check: /actuator/health

6. Application ready
   └─ Listening on localhost:8080
```

## 9. Testing with H2

### Integration Test Example

```java
@SpringBootTest
@DataJpaTest
public class GuaranteeRepositoryTest {
    
    @Autowired
    private GuaranteeRepository repository;
    
    @Test
    public void testSaveAndRetrieve() {
        // Arrange: Create entity
        Guarantee g = new Guarantee();
        g.setGuaranteeNumber("TEST-001");
        g.setAmount(BigDecimal.valueOf(10000.00));
        g.setStatus(GuaranteeStatus.ACTIVE);
        
        // Act: Save to H2 in-memory DB
        Guarantee saved = repository.save(g);
        
        // Assert: Verify H2 stored it correctly
        Guarantee retrieved = repository.findById(saved.getId()).orElse(null);
        assertNotNull(retrieved);
        assertEquals("TEST-001", retrieved.getGuaranteeNumber());
        assertEquals(BigDecimal.valueOf(10000.00), retrieved.getAmount());
    }
    
    @Test
    @Transactional
    public void testFindByStatus() {
        // Insert 2 guarantees
        repository.save(Guarantee.builder()
            .guaranteeNumber("ACTIVE-001")
            .status(GuaranteeStatus.ACTIVE)
            .amount(BigDecimal.valueOf(5000))
            .build());
        
        repository.save(Guarantee.builder()
            .guaranteeNumber("INACTIVE-001")
            .status(GuaranteeStatus.INACTIVE)
            .amount(BigDecimal.valueOf(3000))
            .build());
        
        // Query H2
        List<Guarantee> active = repository.findByStatus(GuaranteeStatus.ACTIVE);
        
        // Verify
        assertEquals(1, active.size());
        assertEquals("ACTIVE-001", active.get(0).getGuaranteeNumber());
    }
}
```

**Nota**: Tests usan el mismo `jdbc:h2:mem:testdb` → datos aislados por test clase.
